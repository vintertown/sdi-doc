# Acronyms

- DNS = Domain Name Server
- IP = Internet Protocol
- URL = Uniform Resource Locator
- dig = Domain information grouper
- CNAME = Canonical name
- A = Address
- AAAA = IPv6 Address
- MX = Mail Exchange
- NS = Name Server
- PTR = Pointer
- SOA = Start of Authority
- BIND = Berkeley Internet Name Domain
